# ![Logo](app/resources/urge_favicon_64.png) Universal Ruby Game Engine (URGE) (WIP)

## Preview

 - URGE is a 3D-based game engine with ruby language supporting.  
 - URGE provides compatitable scripts wrapper for supporting RPG Maker XP/VX/VX Ace.  
 - URGE uses WebGPU specification as the rendering hardware interface backend, which supports modern graphics APIs like D3D12/Vulkan.  
 - URGE can be used on popular platforms like Windows, Linux, Android and Emscripten.
